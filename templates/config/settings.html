Configuration Guide
1 เปิดใช้ Vulnerability Detector จำเป็น

แก้ไขไฟล์ /var/ossec/etc/ossec.conf บน Wazuh Manager

ossec.conf
<vulnerability-detector> <enabled>yes</enabled> <interval>5m</interval> <min_full_scan_interval>6h</min_full_scan_interval> <run_on_start>yes</run_on_start> <provider name="nvd"><enabled>yes</enabled><update_interval>1h</update_interval></provider> <provider name="redhat"><enabled>yes</enabled></provider> <provider name="ubuntu"><enabled>yes</enabled></provider> </vulnerability-detector>
sudo systemctl restart wazuh-manager
2 เปิดใช้ Syscollector บน Agent แนะนำ
ossec.conf บน Agent
<wodle name="syscollector"> <disabled>no</disabled> <interval>1h</interval> <scan_on_start>yes</scan_on_start> <packages>yes</packages> <os>yes</os> </wodle>
3 Custom Integration (Real-time webhook) optional
ossec.conf บน Manager
<integration> <name>custom-soc-dashboard</name> <level>3</level> <alert_format>json</alert_format> </integration>
